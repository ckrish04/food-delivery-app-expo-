# Zaika Mobile (Expo Go)

## Run

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm start
```

3. Scan QR from Expo Go app on your phone.

## Backend URL (Important)

- For Expo Go on a physical phone, `localhost` does not point to your laptop.
- Open **Profile** tab in app and set backend URL to your laptop IP, for example:

`http://192.168.1.10:5000`

- Keep backend running from `backend` folder:

```bash
npm start
```
