# Zaika 

Zaika is a mobile food ordering application developed for college canteens to simplify the food ordering and management process for students and vendors. The application allows users to browse menus, place orders, and track order status in real time through an intuitive mobile interface.

## Features

### Customer Side
- User authentication and login
- Browse food menu and categories
- Add items to cart
- Place food orders
- Real time order status updates
- Responsive and user friendly mobile UI

### Vendor/Admin Side
- Manage incoming orders
- Update order preparation status
- Track completed and pending orders
- Menu management support

## Tech Stack

### Frontend
- React Native
- Expo Go

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Project Architecture

The application follows a client server architecture:

1. React Native frontend handles user interaction and UI.
2. Express.js backend manages APIs, authentication, and order processing.
3. MongoDB stores user data, menu details, and order information.
